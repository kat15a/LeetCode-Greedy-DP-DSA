Problem: Unique Paths

Problem Statement:
A robot is located at the top-left corner of an m × n grid.

The robot can only move:
- Right
- Down

The robot wants to reach the bottom-right corner of the grid.

Task:
Return the total number of unique paths the robot can take
to reach the destination.

Approach:
- Use Dynamic Programming.
- The robot can reach cell (i, j) from:
  - cell (i-1, j) → moving down
  - cell (i, j-1) → moving right
- Therefore:

  dp[i][j] = dp[i-1][j] + dp[i][j-1]

- Initialize the first row and first column with 1 because
  there is only one way to reach those cells.

Algorithm Used:
- Dynamic Programming (Tabulation)

Time Complexity:
- O(M × N)

Space Complexity:
- O(M × N)
