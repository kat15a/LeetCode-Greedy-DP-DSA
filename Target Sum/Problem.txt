Problem: Target Sum

Problem Statement:
You are given an integer array nums and an integer target.

You want to assign either '+' or '-' sign to every number
in the array and then calculate the expression.

Example:
nums = [1,1,1,1,1], target = 3

One possible expression:
+1 +1 +1 +1 -1 = 3

Task:
Return the number of different expressions that evaluate to target.

Approach:
- Let total be the sum of all numbers.
- If we divide numbers into two groups:
  - S1 → numbers with '+'
  - S2 → numbers with '-'

Then:

S1 - S2 = target  
S1 + S2 = total

Solving the equations:

S1 = (total + target) / 2

So the problem becomes:
Count the number of subsets whose sum equals (total + target) / 2.

Use recursion with memoization to count subsets with the target sum.

Algorithm Used:
- Recursion
- Dynamic Programming (Top-Down / Memoization)
- Subset Sum / Knapsack Pattern

Time Complexity:
- O(N × Target)

Space Complexity:
- O(N × Target)
